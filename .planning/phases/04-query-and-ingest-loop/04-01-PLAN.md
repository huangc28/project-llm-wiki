<objective>
Implement the query half of the compounding wiki loop.

Purpose: make project-wiki-query usable without turning the helper script into a final-answer generator. The helper should read `.llm-wiki/index.md` first, expose candidate wiki pages and answer-contract text, and append bounded query log entries after the agent has answered or returned not-covered.
Output: query protocol docs, a real `project-wiki query` helper path, and targeted query fixtures.
</objective>

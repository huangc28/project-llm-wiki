# Vault Today

Build a prioritized daily plan from the vault's current state.

## Steps

1. Read today's daily note if it exists (daily_notes/YYYY-MM-DD.md). If not, create it.
2. Read the last 3–5 daily notes to understand momentum and unfinished threads.
3. Search the vault for any notes modified in the last 3 days.
4. Look for open todos, pending tasks, and flagged items across recent notes.
5. Synthesize into a prioritized plan for today.

## Output Format

**Today's Focus** (1–3 items max — the things that actually matter today)

**Carry-overs** (unfinished threads from recent days)

**Open questions** (things blocking progress that need a decision or answer)

**Optional: one thing to let go of** (something in the vault that's been open too long and should be closed or archived)

Write the plan into today's daily note under a `## Today's Plan` section.

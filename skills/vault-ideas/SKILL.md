# Vault Ideas

Generate actionable ideas by mining patterns across the user's Obsidian vault.

## Steps

1. Use Obsidian MCP to scan recent notes, daily notes, bookmarks, and project notes.
2. Look for:
   - Recurring themes or frustrations the user keeps writing about
   - Half-started ideas that never became projects
   - Domain expertise that could be applied somewhere new
   - Problems mentioned multiple times without a solution
3. Generate 3–5 concrete ideas grounded in what the vault actually contains.
4. For each idea: explain what in the vault sparked it and why it fits the user.

## Output Format

Each idea:
- **Idea**: one sentence description
- **Vault signal**: which notes/patterns inspired it
- **Why it fits you**: based on skills, interests, and context in the vault
- **First step**: the single most concrete next action

Avoid generic ideas. Every idea must be traceable to something specific in the vault.
